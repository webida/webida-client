jQuery Selectors

jQuery selectors allow you to select and manipulate HTML element(s).

jQuery selectors are used to "find" (or select) HTML elements based on their id, classes, types, attributes, values of attributes and much more. It's based on the existing CSS Selectors, and in addition, it has some own custom selectors.

All selectors in jQuery start with the dollar sign and parentheses: $().

The jQuery element selector selects elements based on the element name.

jQuery Events

jQuery’s event system normalizes the event object according to W3C standards. The event object is guaranteed to be passed to the event handler. Most properties from the original event are copied over and normalized to the new event object.

jQuery event methods trigger or attach a function to an event handler for the selected elements.

click()

The click() method attaches an event handler function to an HTML element. The function is executed when the user clicks on the HTML element.

If you click on me, hello popup.

                            $(document).ready(function(){
                                $("p").click(function(){
                                    alert('Hello!');
                                });
                            });

dblclick()

The dblclick() method attaches an event handler function to an HTML element. The function is executed when the user double-clicks on the HTML element.

If you double click on me, hello popup.

                            $(document).ready(function(){
                                $("p").dblclick(function(){
                                    alert('Hello!');
                                });
                            });

hover()

The hover() method takes two functions and is a combination of the mouseenter() and mouseleave() methods. The first function is executed when the mouse enters the HTML element, and the second function is executed when the mouse leaves the HTML element.

If you hover on me, text color changed.

                            $(document).ready(function(){
                                $("p").hover(function(){
                                    // enter
                                    $(this).css('color', 'red');
                                },function(){
                                    // leave
                                    $(this).css('color', '#222222');
                                });
                            });

All of the methods in this section manipulate the DOM in some manner. A few of them simply change one of the attributes of an element, while others set an element’s style properties.
